# 강아지 종 예측하기 
- CNN을 기반 이미지 분류 프로젝트
- 강아지 120종에 대해 2만개의 이미지가 있는 데이터를 활용해 딥러닝 모델을 학습시키고, 임의의 강아지 이미지를 넣었을 때 강아지 종을 정확하게 예측하는 것을 목적으로 함.

- 데이터 : https://www.kaggle.com/datasets/jessicali9530/stanford-dogs-dataset
